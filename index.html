HTML 5 Entities

Beberapa karakter merupakan reserved character di html seperti:

  • <
  • >

Selain untuk menampilkan reserved character html entities juga berfungsi untuk menampilkan karakter-karakter yang tidak ada di keyboard seperti :

  • ©
  • ®
  • §
  • ℃
  • ℉

untuk daftar html entities yang lebih lengkap bisa klik link ini
